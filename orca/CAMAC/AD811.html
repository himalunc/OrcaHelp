AD811 ADC

 

This is an 8-channel analog-to-digital converter with 11-bit resolution. It is triggered by an external strobe. It is made by Ortec. This card can exist only in a CAMAC crate.

 

Test functions for card

Enables each of the four channels

 
 

ADC Data Format


The card will use the short form if at all possible. However, you must be prepared to decode both types.


Short Form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^--------------------------------- AD811 ID (from header)

--------^-^^^--------------------------- Crate number

-------------^-^^^^--------------------- Card number

-----------------------^---------------- Channel number

-------------------------^^^^ ^^^^ ^^^^- adc value


Long Form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^----------------------- AD811 ID (from header)

-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^- length (always 2 longs)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

--------^-^^^--------------------------- Crate number

-------------^-^^^^--------------------- Card number

-----------------------^---------------- Channel number

-------------------------^^^^ ^^^^ ^^^^- adc value
