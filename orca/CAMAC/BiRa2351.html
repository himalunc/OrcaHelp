BiRa 2351

 
The Model 2351 is a single-width module that provides a 12-bit register for holding NIM input data. The 12 NIM inputs and NIM Input Strobe via front panel connectors. LAM status is set at leading edge of the strobe. If enabled and LAM status is set, a LAM requested is generated.

Data inputs must precede the strobe by at least 10 ns. The strobe signal must be a minimum of 10 ns wide. All inputs are active low (-12 to 36mA). An open input is a logic 0 level.

 

Set up the polling interval

Enabled channels will appear in the data read out list with the label that is specified in the dialog for that channel. If an event is triggered, each of the items in the read out list for each triggered channel will be read out.
