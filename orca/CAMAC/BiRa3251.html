BiRa 3251

 
The Model 3251 is a single-width module that provides a 12-bit register , NIM level outputs. The register is loaded from the dataway and outputted via front panel LEMO connectors. A strobe is generated when the register is loaded. The strobe duration is adjustable. Straps are provided for user determined durations of 50 ns to several seconds by the addition of timing capacitors.

Terminators are required to produce a true output, -0.8 to -1.2 VV, false output is 0.0V.

The strobe may be internally strapped to gate one or all channel groups 1-4, 5-8, 9-12 to be pulsed outputs. If strap points are open the output groups are DC levels.

 

Enabled channels will appear in the data read out list with the label that is specified in the dialog for that channel. If an event is triggered, each of the items in the read out list for each triggered channel will be read out.
