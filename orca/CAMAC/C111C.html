C111C Controller

 

Connect/disconnect from the host at the indicated IP address

Select NAF command to send to controller. Note the command description area.

This is a commercial card designed to serve as the controller for a CAMAC crate. This card can exist only in a CAMAC crate and must be placed in the last slot of the crate. Communication to the ORCA host is via an Internet socket using TCP-IP. See the manual for complete information. There is also a programmer’s guide that has a complete description of the API used by the communication library.


Note that this device is quite slow.
