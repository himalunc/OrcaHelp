Joerger ADC AM-F

 
The Joerger Enterprises, Inc. Model AM-F is a scanning ADC and uses a successive approximation ADC providing a cycle time of less than 40 µsec. The standard input range is +/-10V DC, and with a programmable range option the ranges are +/-10V, +/-5V, and +/-1V. Unipolar inputs are also available. The input range can be changed in the field by an on-board jumper or ordered with any range from 1 volt to 10 volts. Current inputs of 4-20mA can also be accepted.

For a complete description, please refer to the manual.

 

Data Records (NOT!)


The JAM-F is not officially an ORCA data taker. Thus, there are no data records associated with this hardware object at this time. This card can participate in the Process Controller as an ADC element.
