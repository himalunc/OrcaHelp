CAMAC Crates

 

There are two kinds of CAMAC crates in the catalog, a 13 slot LeCroy8013 crate and a 25 slot standard crate. The icons in the catalog look like this:

Helpful Hint.


Selected cards can be moved left and right using the arrow keys.


You can make duplicate cards using copy/paste. All parameters will be copied, so if you need more than one complex card, set up one card and copy/paste the rest.

Double-click cards to open their dialogs

Card labeling option

Crates require some kind of controller (i.e CC32) to function. It always goes into the last two slots.

Drag and drop CAMAC card objects from the catalog into empty slots

Click to disclose addition control functions

The crates will not allow multiple cards to occupy the same slot and each card must be in the same slot in the configuration as in the actual physical crate.
