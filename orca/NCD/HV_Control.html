HV Control

 

The High Voltage Control object controls, monitors, and displays all HV parameters for the NCD experiment. This is an object specific to the NCD experiment and works only in conjunction with the Mux controller.

 

Enter the values of the HV supply slope and offset to have the read back values correctly match actual values.

HV relay control for HV supplies tat are enabled (only possible when HV is at zero Volts.

Ramp controls for HV supplies that are enabled

Check which supplies to control, enter the ramp up time and the desired voltage. Displays the voltage currently being written (DAC), actual current (nAmps), and state of supply (On, Off, Ramping Up, Ramping Down, Panic)

Set interval between HV parameter checks and display of values

Full Panic of all HV (enabled when at least one supply is on)

Set location where HV parameters stored

Optionally save the currents to file

Strip chart of the HV currents over time.
