IP Fifo

 
The IPFifo is a memory card that was used during the development of ORCA to test PCI-VME communications and other low-level code. It has not been developed into a useful object.  The IPFifo object can only be placed onto an IP Carrier board.
 

The Fifo card dialog has some low-level test functions

If a need arises for an actual use for this card, contact one of the developers with your needs.
