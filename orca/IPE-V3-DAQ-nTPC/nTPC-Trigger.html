Pixel Trigger:

The pixel trigger runs a sum filter (boxcar filter) with length L over the ADC trace. If the sum is greater than a given threshold then the pixel trigger is set to 1.

Multiplicity Trigger:

The SLT card receives all pixel trigger flags of all channels in all FLT cards. If more than N channels have triggered in the same time window (length adjustable) a multiplicity trigger is generated and a event data record is written out.
