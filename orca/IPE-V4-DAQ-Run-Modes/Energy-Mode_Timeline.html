Timing in Energy(+Trace)-Mode:

The SLT sends the second strobe over the backplane in order to provide the same time base for all cards of the crate. When ORCA gets the Start Run command, it needs to send all settings (gains, thresholds,...) to the crate. After that the measurement cycle is started by sending a ‚release inhibit‘ to the crate. When ORCA stops the run, it will take some delay until the crate receives the ‚set inhibit‘. The Run Time Counter is running during the crate is not inhibited so it provides accurate information about the timing.
