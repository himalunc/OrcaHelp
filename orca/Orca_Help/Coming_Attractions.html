Coming Attractions



Orca is constantly under development. This page is meant to serve as a location to list the things that are under development and will be released in the near future.




XCode 9. Orca is still a 32-bit application. An experimental version exists.


XCode 5. Orca should run normally under XCode 5. There are a number of compiler warning about depreciated methods. You can safely ignore them. They will be cleared as time permits.


XCode 4 Support.  XCode 4 no longer supports InterfaceBuilder plugins. Many, many dialogs in ORCA use plugins. The code has been swept and plugin objects have been replaced with new objects. This means the dialogs that use things like plots, axis, queue bars, rate bars, etc are no longer WYSIWYG and just show blank NSViews during the editing process. It is thus very difficult to line objects up with their counterparts, i.e. getting an axis lined up with a plot. A number of composite objects are being developed that at least make life easier for the dialog developer.


CouchDB and mySQL support.  In order to support remote monitoring via web applications, an interface to a couch database is being added. Currently the run status, alarms, and data histograms are inserted into a database if one of the (undocumented) database objects are in the configuration.


Feb ’10. The plotting subsystem is being re-factored.  Lately there have been some instabilities associated with the plots and the analysis subsystem. A redesign of the plot/analysis object graph will add new features and attempt to address these problems.


SIS3302 card development underway! (Finished)


ORCA is now runs under 10.6! 10.5 will continue to be supported for some time.


ORCA is now 10.5 only! I will no longer support 10.4 and ORCA is now a 10.5 only application. This is necessary in order to take advantage of the new technologies that are available only in 10.5.


KATRIN PAC. The Power and Control module is ready for testing.


CENPA HV controller. Custom built HV controller built around a generic ADC/DAC controller.


CENPA integrating pico-Amp meter. Ready for testing as of 4/22/07.


CAEN 1720 Digitizer. Support for this VME module is being added.


KATRIN Experiment. Code to help support the KATRIN neutrino mass measurement experiment is under active development, in particular the version 4 SLT and FLT objects.


Majorana Experiment. Objects to support the Majorana double beta decay experiment are under development.


SNO+ Experiment. Objects to support the SNO+ experiment at SNOLab at Sudbury, Ontario are under development.
