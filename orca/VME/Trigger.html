Trigger Card

 

Set the VME Address

Reset status register, clock, read board ID, and prints status register contents

This card was built at the University of Washington to interface between the SNO Master Trigger Card and the NCD System (shapers and muxes).


This card can only be placed in a VME Crate Object.
