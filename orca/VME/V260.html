CAEN V260

 
The Mod. V260 is a single width VME module housing 16 independent 24 bit counting channels at the maximum input frequency of 100 MHz.

Each channel can be enabled to generate an interrupt signal when the counter reaches 2^16 or 2^24 (selection via internal jumpers). The status of the internal jumpers can be read via std. VME read cycles.


The unit exists in three versions that accept ECL (V260 E), NIM (V260 N) or TTL (V260 T) inputs.


Up to 16 channels can be cascaded for a total of 24 x 16 bit counting depth.


Please refer to the manual for more details.
