IP Carrier

 

Set the VME Address. IP cards will be placed at increments os 0x100 from this address.

This card is a carrier for up to four standard Industry-Pack modules. The IP Carrier object can only be placed into a VME crate object. Double-clicking it brings up a dialog onto which you can drop one of the IP modules, i.e. the IP408. The base address jumper settings and other information can be found in the manual.
 

Locations to drop IP cards

Here's an example of a carrier that is holding one IP module, the IP408. IP modules can be dragged to different positions on the card. When an IP module is inserted into the carrier, a connection point will appear on the IP carrier card in the VME crate view.
