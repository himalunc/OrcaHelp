32 Bit Trigger Card

 

Set the VME Address

Sets Triggers names that are displayed in the Readout list.

Read status and board info (displayed in the status log window)

This card was built at the University of Washington to interface between the SNO Master Trigger Card and the NCD System (shapers and muxes).


This card can only be placed in a VME Crate Object. For more information on the Trigger 32-Bit Card see the manual.

Global Trigger Record

The trigger card will put out the short form if at all possible, but will switch to the long form if it has to.


Short Form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^---------------------------------GTID Record ID

-------^^-------------------------------event trigger bits

---------------^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid for this trigger event


Long Form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^-----------------------GTID Record ID

-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length (always 2 longs)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

-------^^-------------------------------event trigger bits

---------------^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid for this trigger event


Live-Time Record

The trigger cards can put out the following record which is fixed length. The format is:


xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^-----------------------record id

-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-size including this word (in longs)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

----------^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

-----------------^^---------------------1,0,3 startrun,stoprun,middlerun

-------------------------^^^^-----------crate

---------------------------------^ ^^^^-slot

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^--------------------------------top 8 bits of the scope livetime

----------^^^^ ^^^^ ---------------------top 8 bits of trigger2 livetime

--------------------^^^^ ^^^^ -----------top 8 bits of trigger1 livetime

------------------------------^^^^ ^^^^ -top 8 bits of total livetime

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of total livetime

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of trigger1 livetime

xxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of trigger2 livetime

xxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of scope livetime
