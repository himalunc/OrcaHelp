CAEN V419

 

Set the VME Address

The Mod. V419 is a 1-unit wide VME 6U module which houses 4 channels of linear gate and stretcher with a multiplexed 12 bit ADC.
The input signals can be square waves, or semi-gaussian positive pulses with a rise time greater than 0.5 µs; it can also convert constant or very slow signals working in sampling mode. A sliding scale technique is featured to improve the differential non linearity.
The unit has an A24, D16 VME interface.


Please refer to the manual for more details.

 

V419 Data Record format


The data records for the CAEN cards is exactly as produced by the cards with the addition of a header that 'wraps' the data for ORCA.  The format is:


ADC record short form:

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^--------------------------------- ID (from header)

--------^-^^^--------------------------- Crate number

-------------^-^^^^--------------------- Card number

--------------------^^^^---------------- Channel number

-------------------------^^^^ ^^^^ ^^^^- adc value


ADC record long form:

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^----------------------- ID (from header)

-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^- length (always 2 longs)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

--------^-^^^--------------------------- Crate number

-------------^-^^^^--------------------- Card number

--------------------^^^^---------------- Channel number

-------------------------^^^^ ^^^^ ^^^^- adc value
