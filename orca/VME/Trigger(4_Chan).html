4 Channel Trigger Card

 

Set the VME Address, defaults to 0x7500, which is currently hardcoded in the board hardware

Reset status register, clock, read board ID, and prints status register contents

This card was built at the University of Washington to provide up to 4 different triggers (for VME cards) and is in use by the UW Radiology lab and the KATRIN experiment.


This card can only be placed in a VME Crate Object.

 

Trigger Clock data record format:

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^-----------------------Clock record ID

-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length


xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

------^^^-------------------------------event trigger name

----------^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-top-mosts bits of 50 MHz time

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx

^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of 50 MHz time

Names can be given to each trigger. These will appear in the dat readout task.

Enables the clock (must be checked to ship timing records) and controls to enable the shipping of each event clock -- one for each trigger. The ship first and last clock option will ship the the first and last clock regardless of the other ship opions

Load specific values into top and bottom clock register. Clock reset, enable and disable. Note: it is a 50 MHz clock.

Reads current value in the clock register. Latches clock into software register.
