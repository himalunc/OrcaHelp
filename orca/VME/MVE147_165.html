MVE 147/165 eCPU

 

Shows the progression of the data being put into and read out of the data circular buffer

Orca provides support for using a VME-based embedded CPU. Either the MV147 or the MV167 can be used. The include download code included in the distribution only supports eCPU readout of the shaper ADC cards at this time. That code may be expanded to support other cards in future releases of ORCA. Custom code can be written to support other cards using the existing download code as an example, but the current compiler restriction to the Think-C compiler means that for all practical purposes this card is obsolete as far as ORCA is concerned. Data is readout to ORCA using the dual-port memory of the controller card, either the 617 or the 620


This object can only be placed in a VME type crate.
